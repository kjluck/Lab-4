Each color is assigned an RGB value, a value written in hexadecimal notation, and in HSL. The coder publishes CSS to color and style websites. The coder can use these numbers when coding a style sheet.

Preferred color code format: hexidecimal format.

HWB stands for hue, whiteness and blackness. Like HSL, the hue can be anywhere within a range of 0 to 360. The other two arguments control how much white or black is mixed into that hue, up to 100% (which would result in a totally white or totally black color).

CMYK color stands for cyan (C), magenta (M), yellow (Y), and black (K for “key”), and is the color mode used by commercial printing equipment to create full-color graphics and images.
